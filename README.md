This is a practice project focused on **Neural Decoding**, aiming to explore neuroscience data using Python (or matlab) and various computational algorithms.

## Quick Start

Follow these steps to set up and run the project in your local environment.

### 1. Prerequisites
* Python 3.10 or higher installed.
* Git installed and configured.

### 2. Installation & Environment Setup
First, clone the repository and set up a clean virtual environment:

```bash
# Clone the repository
git clone git@github.com:YunAnGitHub/neural_decoding_funpark.git
cd neural_decoding_funpark

# Create a virtual environment (venv)
python -m venv .venv

# Activate the virtual environment
# For Windows:
.\.venv\Scripts\activate
# For macOS/Linux:
source .venv/bin/activate
